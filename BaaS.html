What is Unitellas Backup-as-a-Service?

A method of data storage in which files, folders, or the entire contents of a hard drive are regularly backed up by a service vendor to a remote secure cloud-based data repository over a network connection.

One major benefit of using our compute service is intelligent load balancing. A load balancer allows you to automatically distributes incoming traffic across multiple targets, such as instances, or IP addresses.

There are two types of load balancers implemented by Unitellas:

Application Load Balancer (ALB) - distributes HTTP or HTTPS traffic using application layer protocol based routing.

Load Balancer (NLB) - Distributes TCP traffic regardless of the application layer protocol.

If it (the load balancer) finds an overloaded node(device), it will automatically move workloads away from that node to a node with more capacity - typically moving a virtual machine from source to destination in about half a second.
